# Screen Explorer Client

Angular Application.

## Commands

Serve project in development.

```bash
npm start
```

Format project.

```bash
npm run format
```

Format project with watch mode.

```bash
npm run format:watch
```

Lint project.

```bash
npm run lint
```

Test project.

```bash
npm test
```

Test project without watching files.

```bash
npm run test:without-watch
```

E2E project.

```bash
npm run e2e
```

Analyze project.

```bash
npm run analyze
```

Build project for production.

```bash
npm run build:prod
```

Build project for production with Angular Universal (SSR).

```bash
npm run build:ssr
```

Build project for production with Prerender.

```bash
npm run prerender
```
